1. 계정 정보 등록
 git config --global user.name '사용자명'
 git config --global user.email '사용자이메일'

2. git-prac 폴더를 git이 관리하는 폴더로 지정
 git init

3. 기본으로 생성되는 master 브랜치에서 main 브랜치로 변경
 (옛날에는 master, slave 로 사용하다가 정치적으로 문제가 되서 main으로 변경했다고 함.)
 git checkout -b main : main 브랜치를 생성하고 해당 브랜치로 이동
 git branch -M main : 기본브랜치를 main브랜치로 지정
 git status : 상태확인

4. 파일을 생성하고 add, commit 하기
 git add <파일명> : 파일을 staging area 에 추가.
 git commit : commit 에 대한 메세지를 남기기 위해 vi 에디터가 실행
		i :  입력모드로 변경, esc : 입력모드에서 나오기, w 저장, q종료 , wq 저장 종료
 git commit -m 'message'  :  commit 을 하고 commit 에 대한 간단한 메세지 작성
 git log : commit 내역 확인
